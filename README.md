# OkGntpd

An GNTPd which always return "OK". Sorry, *this is not yet daemon*.

## Installation

Add this line to your application's Gemfile:

    gem 'ok_gntpd'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ok_gntpd

## Usage

Start GNTP server with:

    $ ok_gntpd

Press Ctrl-C to stop it.
